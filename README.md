# eve_industry_manager

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn electron:serve
```

### Run the backend server
```
yarn serve:server
```

### Run both the electron and backend server in one command
```
yarn dev
```

### Compiles and minifies for production
```
yarn electron:build
```

### Lints and fixes files
```
yarn lint
```
